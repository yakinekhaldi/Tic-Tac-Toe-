# Tic Tac Toe — Terminal

A simple tic tac toe game in the terminal. You play as **X**, the AI plays as **O**.

The AI isn't random — it tries to win, blocks you if it has to, and takes the center when it can.

## Run

```bash
python tictactoe.py
```

No installs needed, just Python.

## How to play

- Enter a number from 1–9 to place your move
- The board positions match a numpad layout:

```
1 | 2 | 3
--+---+--
4 | 5 | 6
--+---+--
7 | 8 | 9
```
